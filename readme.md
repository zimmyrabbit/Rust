RUST BUILD SYSTEN 'CARGO'

cargo new [project name]    : 프로젝트 및 템플릿 생성 <br/>
cargo build                 : 프로젝트 빌드 <br/>
cargo run                   : 프로젝트 실행 <br/>
cargo check                 : rust 문법 체크 <br/>
cargo test                  : 프로젝트 테스트 <br/>
cargo doc                   : 문서 생성 <br/>
cargo publish               : 라이브러리 공개 <br/>
