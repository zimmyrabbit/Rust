RUST BUILD SYSTEN 'CARGO'

cargo new [project name]    : 프로젝트 및 템플릿 생성
cargo build                 : 프로젝트 빌드
cargo run                   : 프로젝트 실행
cargo check                 : rust 문법 체크
cargo test                  : 프로젝트 테스트
cargo doc                   : 문서 생성
cargo publish               : 라이브러리 공개
